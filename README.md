# Admission Verification System
Beginner Python project using nested if-else conditions.

## Features
- Validates registration number, stream, marks, and password
- Grants or denies admission based on rules
- Demonstrates nested if-else logic in Python

## How to Run
1. Install Python (3.x recommended).
2. Save the file as `admission_verification.py`.
3. Run in terminal:
   ```bash
   python admission_verification.py
